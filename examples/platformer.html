pIvotX — Platformer

A complete platformer demo using Tilemap, Camera, TiledBackground (parallax), SpriteAnimator, and AABB collision. Collect all coins to win!

Features demonstrated

• AssetLoader.loadAssets()        — batch preloading
• Tilemap + solidTiles            — grid-based level with collision queries
• getSolidTilesInRegion()         — broad-phase tile collision
• aabbOverlapDepth() + createAABB — per-tile collision resolution
• Camera.follow() + clamp()      — smooth scrolling camera
• TiledBackground                 — parallax sky + hills layers
• SpriteAnimator                  — idle / run / jump clips
• Sprite (coin-sheet)             — animated collectibles
• GameImage                       — static images

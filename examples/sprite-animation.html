pIvotX — Sprite Animation

Demonstrates AssetLoader, Sprite, SpriteAnimator, and GameImage. Use ← → arrow keys to run. Press Space to jump.

What this demo shows

• AssetLoader.loadAssets() — batch-preloads all images before the loop starts
• Sprite.createSheet()    — creates a SpriteSheet from a loaded image
• SpriteAnimator          — manages idle / run / jump clips by name
• GameImage               — draws a static background image
• Sprite.flipX            — mirrors the character when facing left
